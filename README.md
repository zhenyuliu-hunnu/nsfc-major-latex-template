# nsfc-major-latex-template

国家社科基金重大项目 LaTeX 模板，适用于年度重大项目和重大专项。

模板参考：http://www.nopss.gov.cn/n1/2025/0526/c431028-40488013.html

Tip：LaTeX 的左引号是``，右引号是 "。

## 使用说明

本项目仅包含重大项目的第三和第四部分（表 4～表 10），属于申报书最主要的内容，请在官网下载对应 Word 文档，填写其他部分，导出 PDF 后与此项目编译出的 PDF 进行合并。

参考工作流程：

1. 用 LaTeX 写正文（主内容与子课题）
2. 处理参考文献
3. 用 Word 填写其他内容
4. 根据 Word 页码填写 LaTeX 文件起始页码
5. 在 Word 中填写页码，包括目录中的页码，第五部分及之后几页的页码（自行修改）
6. 用 LaTeX 编译 PDF、Word 导出 PDF
7. 使用 PDF 编辑软件，将 LaTeX 编译的 PDF 插入到 Word 导出 PDF 的对应位置

## 文件结构说明

`main.tex` 用来编译的文件

`table_4.tex`~`table_10.tex` 目录中表 4 - 表 10 的内容

`sub1.tex` 子课题一的内容	

`sub2.tex` 子课题二的内容，如还有其他子课题，请复制这个文件并自行更名并引入 `main.tex`

`reference.bib` BibLaTeX 文献库，用来进行内文的文献综述引用，但请不要使用 BibLaTeX 生成表 9 的引文目录

## 本模板使用的格式

最终格式预览请查看 `main.pdf`文件

- 一级、二级、三级标题：均为楷体小四号字加粗，行距 22pt，首行缩进 2 中文字符。每一个表格中的标题序号都从 1. 开始，换而言之下一个表格不会承接上一个表的章节编号
- 正文：楷体小四号字，行距 22pt，首行缩进 2 中文字符
- 图片编号：不分章节，按顺序进行编号，如`图1: 内容`
- 公式编号：没有做公式相关，欢迎 PR

## 编译参考环境

- macOS 26.2
- MacTeX - 08 March 2025
- XeLaTeX + BibLaTeX

本项目未在 Windows 和 Linux 上进行编译，不免会遇到问题，请自行排查。
